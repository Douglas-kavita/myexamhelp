  <h1 style="margin:0 0 8px 0;">AWS Certification Overview (2026 Guide)</h1>
  <p style="margin:0; color:#334155;">
    AWS certifications are <strong>role-aligned, standardized exams</strong> built from a <strong>Job Task Analysis (JTA)</strong> and official test specifications.
    The goal is to sample the skills required for a role — <strong>not</strong> to test everything you could ever learn.
  </p>

  <div style="margin-top:12px; background:#eff6ff; border-left:6px solid #60a5fa; padding:12px 14px; border-radius:12px;">
    <strong>AWS certifications are NOT:</strong>
    <ul style="margin:10px 0 0 18px;">
      <li>A guarantee of job readiness by themselves (hands-on + portfolio still matters).</li>
      <li>A replacement for real experience (AWS recommends hands-on work aligned to each exam outline).</li>
      <li>One “AWS mastery” badge (each exam targets a specific role/skill track).</li>
    </ul>
  </div>
</div>

  <p style="margin:0; color:#334155;">

  </p>

  <div style="margin-top:12px; background:#f0f9ff; border-left:6px solid #60a5fa; padding:12px 14px; border-radius:12px;">
    <ul style="margin:0 0 0 18px;">
      <li><strong>CloudOps Engineer – Associate</strong> replaces <strong>SysOps Administrator – Associate</strong> (SysOps retirement completed in 2025).</li>
      <li><strong>Security – Specialty (SCS-C03)</strong> is the current security specialty version.</li>
      <li><strong>Machine Learning – Specialty</strong> is announced for retirement by <strong>March 31, 2026</strong> (plan accordingly).</li>

      <li>Previously retired certs (confirmed by AWS announcements): <strong>Data Analytics – Specialty</strong>, <strong>Database – Specialty</strong>, <strong>SAP on AWS – Specialty</strong>.</li>

  <h1 style="margin:0 0 10px 0;">Eligibility & Requirements (Location-Specific)</h1>
  <p style="margin:0; color:#334155;">
    AWS has <strong>no prerequisites</strong> for certifications, but it does publish experience recommendations per exam.
  </p>

  <div style="margin-top:12px; background:#f1f5f9; border-left:6px solid #94a3b8; padding:12px 14px; border-radius:12px;">
    <strong>Key points:</strong>
    <ul style="margin:10px 0 0 18px;">
      <li>Candidates <strong>13–17</strong> may test with <strong>parent/legal guardian consent</strong>.</li>
      <li><strong>Name matching is strict:</strong> your AWS Certification account name must match your ID(s), or you can be denied entry.</li>
      <li>ID rules differ by mode and country: primary/secondary IDs, unacceptable IDs, local-language rules.</li>
      <li>Online proctoring can be restricted in some locations (provider restrictions).</li>
    </ul>
  </div>
