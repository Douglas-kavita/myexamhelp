## A) What Praxis is (and is not)

<div style="background:#f0fdf4; border:1px solid #34d399; padding:14px; border-radius:14px; margin:12px 0;">
  <b>What Praxis is (ETS program + measurement):</b>
  <ul style="margin:8px 0 0 18px;">
    <li>Praxis is a suite of licensure-related assessments developed and administered by ETS.</li>
    <li>Used by states/agencies to evaluate job-relevant academic skills, subject knowledge, and/or pedagogy knowledge for safe, effective beginning practice.</li>
    <li>Computer-delivered for essentially all candidates (rare exceptions exist, e.g., Braille Proficiency 0633 uses a separate registration process).</li>
  </ul>

  <div style="height:10px;"></div>

  <b>What Praxis is NOT (state licensure + employer discretion):</b>
  <ul style="margin:8px 0 0 18px;">
    <li>Not a teaching license.</li>
    <li>Not a guarantee of employment.</li>
    <li>Not a single test: there are multiple test families and hundreds of test codes.</li>
    <li>States and educator preparation programs (EPPs) use Praxis as one component; employers can add requirements.</li>
  </ul>
</div>

### Praxis test families (Core, Subject Assessments, PLT, and related)
ETS uses multiple categories (states choose what they require):
- **Praxis Core Academic Skills for Educators**: reading, writing, mathematics  
- **Praxis Subject Assessments**: subject/specialty tests (some are multi-subtest)  
- **PLT (Principles of Learning and Teaching)**: pedagogy, grade-band specific  
- **Content Knowledge for Teaching (CKT)**: elementary-focused content used in teaching (often in subtests)  
- **School Leadership Series (SLS)** and other related assessments appear in ETS technical documentation as part of the broader “Praxis Tests and Related Assessments” ecosystem

### How Praxis fits into licensure + endorsements
**State licensure policy (the key point):** States/agencies decide:
- Which Praxis tests are required for each license/endorsement  
- What the passing/qualifying score is  
- Whether Praxis is required at all or whether alternatives are accepted (state-specific tests, coursework routes, portfolio/performance assessments, etc.)

**ETS policy (candidate-facing):** ETS provides tools to:
- Find state/agency requirements  
- Register, schedule, test at center or at home (when available), and report scores

### Common misconceptions (correct framing)
- “ETS sets passing scores.” → **Wrong.** Passing/qualifying scores vary by state/agency and are **not set by ETS**.  
- “If I pass Praxis in one state, I’m automatically good everywhere.” → **Not automatic.** Reciprocity is state policy; cut scores can differ.  
- “At-home is easier/different.” → ETS states at-home tests are **identical in content and on-screen experience** and are proctored live.

### Age/education requirements + background checks
- ETS generally does not frame Praxis as requiring a specific age/degree to sit; ETS directs candidates to check state requirements and follow the licensure pathway.
- Background checks, fingerprints, degree/EPP requirements are state-controlled and vary.

### How to verify (no guessing)
1) Open ETS Praxis **State Requirements** page → select state/agency → find your certificate/endorsement area  
2) Open your state DOE/licensure board site → confirm:
- eligibility (degree/EPP status)
- background checks/fingerprinting
- substitutions/waivers
- deadlines and validity windows

## ID requirements & name-matching rules (ETS policy; high-stakes)

<div style="background:#fff7ed; border:1px solid #f97316; padding:14px; border-radius:14px; margin:12px 0;">
  <b>Core ETS rules (must follow):</b>
  <ul style="margin:8px 0 0 18px;">
    <li>Bring acceptable, valid, <b>unexpired physical ID</b> with name + signature + recognizable photo.</li>
    <li>No copies; <b>no ID on a phone</b>.</li>
    <li>As of Jan 1, 2023, ETS no longer accepts expired driver’s licenses.</li>
    <li>Your first and last name must match exactly between registration and ID (including two-part last names).</li>
    <li>If your name changed after registering, ETS indicates you may need to cancel, create a new account, and re-register under the correct name.</li>
  </ul>
</div>

## Accommodations (ETS Disability Services) — types, process, timelines, risks

**Policy fundamentals**
- Request and receive approval **before** scheduling.
- Some accommodations can be self-scheduled; others require coordination.
- Review timeline is commonly **~4–6 weeks** (plan ahead).

**Accommodation process (step-by-step)**
1) Create/log into Praxis account  
2) Start accommodations request via ETS Disability Services + Bulletin Supplement workflow  
3) Submit documentation aligned to ETS documentation guidelines  
4) Wait for approval → schedule → keep PDF records of approvals

**Risk management**
- Most preventable problems: incomplete documentation + late submission.
- If accommodations are mission-critical, submit early and avoid last-minute test dates.

## Special cases (international candidates, name changes, PLNE, test availability)

- **International testing:** ETS notes Praxis is administered through an international network of test centers, including Prometric and Strategic Testing Networks (STNs).
- **At-home availability limits:** Praxis at-home is for test takers living in the U.S., its territories, or Canada (based on your account address).
- **PLNE (Primary Language Not English):** Bulletin includes PLNE forms; ETS indicates processing-time planning (e.g., “allow approximately three weeks” for some requests). Follow Bulletin instructions precisely.

## C) Exam Blueprints & Skills (Praxis-correct)

<div style="background:#eef2ff; border:1px solid #818cf8; padding:14px; border-radius:14px; margin:12px 0;">
  <b>Non-negotiable accuracy rule:</b>
  <ul style="margin:8px 0 0 18px;">
    <li>For every Praxis test code, the single source of truth for domains, weights, question types, and time is the <b>current ETS Study Companion / “Test at a Glance”</b> for that exact code.</li>
    <li>ETS explicitly points candidates to the Study Companion for duration/details.</li>
  </ul>
</div>

### Praxis Core (academic skills)
**Skills tested**
- Academic reading comprehension and analysis  
- Standard written English + composition under time constraints  
- Math reasoning aligned to college/career-ready skills (and calculator rules where applicable)

**Core Writing (5723) blueprint example (official structure)**
- Time: **100 minutes** total → 40 minutes selected-response + two 30-minute essays  
- Questions: 40 selected-response + 2 essays  
- Content categories/weighting:
  - Text Types, Purposes, and Production (includes essays): ~60%  
  - Language and Research Skills for Writing: ~40%  
- Psychometric detail: unscored (pretest) questions may appear; category counts may vary by form.

**Common Core question traps (archetypes)**
- “Best revision” vs “identify error” → changes can be grammatical but harm logic/clarity  
- Research skills → mixing up what’s asked (credibility vs relevance vs citation)  
- Essays → over-reading/over-editing instead of thesis + evidence + structure

## Praxis Subject Assessments (content knowledge + specialty areas)

**What they include**
- Subject/specialty tests (discipline content)  
- Multi-subtest structures (some licenses)  
- CKT in some elementary pathways  

**How to read any Study Companion (universal method)**
1) Find “Test at a Glance” → time, number of questions, domain weights  
2) Identify question types:
   - selected-response  
   - constructed-response  
   - listening/speaking (world languages)  
   - multi-subtest bundles (state-specific pass rules)  
3) Build your study plan from the “Content Topics” sections

**Representative archetypes (descriptions)**
- “Best answer” conceptual items (not trivia)  
- Scenario/application items (especially CKT-style)  
- Multi-step problems with distractors tied to common misconceptions

**New state-required variants to watch (2026+ change risk)**
- ETS is launching new **Praxis Elementary Education Fundamentals** tests in Spring 2026 designed to eventually replace existing series (including **5001, 7001, 5901, 7811**) and introduce a **Praxis Steps** modular approach (state adoption varies).
- If you’re pursuing elementary licensure: confirm whether your state accepts current series or is transitioning.

## E) Scoring & Passing

### How scores are reported (by test family)
- Many Praxis tests report scaled scores; score reports explain performance ranges and measurement error.
- Some tests combine selected-response and constructed-response; reports may show component performance (e.g., Core Writing SR vs essays).

### Raw → scaled (why it matters)
- Scaled scores make forms comparable.
- Measurement error exists; score reports reflect statistical properties.

### Passing scores are state-set (how to find them)
- ETS provides a Passing Score Requirements tool and states clearly that passing scores vary by state/agency and are not set by ETS.
- States may require different tests/cut scores for the same endorsement area.

### Score reporting timelines + sending scores
- ETS provides “Getting Your Praxis Scores” tool with report dates.
- During registration, choose up to **4 score recipients free**, and you can change them for free up to **3 days** before test date.
- Additional score reports cost **$50 per recipient**.
- Praxis scores are valid for **10 years for ordering additional score reports** (within that reporting window).

## H) Preparation Strategy (Beginner → Elite)

### Evidence-based principles (high ROI)
- **Practice testing (retrieval practice):** testing yourself improves long-term retention more than re-reading  
- **Spaced practice:** distributed study beats cramming  
- High-utility learning techniques consistently rate **practice testing** + **distributed practice** above many popular “feel-good” methods

### Diagnostic plan aligned to your exact test code
**Step 1: Lock test code + version**
- Confirm exact code(s) from state requirements + ETS listing

**Step 2: Use official structure to build diagnostics**
- Download Study Companion → create diagnostic checklist from content categories
- If available, take ETS interactive practice test for baseline map

**Step 3: Build an error log (engine of improvement)**
Tag every miss by:
- content domain  
- skill type (concept / procedure / interpretation / writing mechanics / pedagogy principle)  
- error type (misread / knowledge gap / pacing / trap answer / etc.)

## K) Test-Day Strategy & Anxiety Control

**Sleep/nutrition**
- Avoid night-before cramming; you want stable recall under time.

**Pacing and triage**
- minutes per question = total minutes ÷ number of SR questions
- Mixed-format: reserve time for essays/CR first, then back-calc SR pacing
- Two-pass SR strategy:
  1) answer fast/medium items  
  2) return to flagged  

**Guessing strategy (ethical + practical)**
- If no penalty and unanswered counts wrong, unanswered is almost always worse than best-effort guessing. Confirm in Study Companion/on-screen rules.

**If tech fails / proctor issues**
- If test not taken due to check-in/technical issue, ETS indicates you may re-register within 3 business days and it does not count as a retake (28-day rule doesn’t apply).
- For at-home proctor issues: follow test-day support instructions and use formal complaint process if needed.

**Anxiety technique (research-supported)**
- Expressive writing: brief writing about worries immediately before an exam can improve performance for high test-anxious students.

## L) After the Test: Licensure Next Steps

**Sending scores**
- Up to 4 free recipients during registration; can change for free up to 3 days before test.
- Additional reports cost $50 per recipient, available to order for 10 years.

**Interpreting results + retake decisions**
- ETS retake rule: **28-day wait** before scheduling the same test again.
- Retake helps when you missed by small margin and error log shows fixable patterns.
- Retake wastes time when you lack foundations or you’re taking wrong test code.

**Timelines vary by state**
Use: ETS state requirements → state licensure site for application timeline, background checks, EPP completion/student teaching, alternative routes, reciprocity.
